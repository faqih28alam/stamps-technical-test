1. Small Program (FizzBuzz Variation)
A logic sequence from 100 down to 1 with the following rules:

Reverse Order: Iterates from 100 to 1.

Prime Filter: Skips and does not print any prime numbers.

FizzBuzz Logic: * Replaces multiples of 3 with "Foo".

Replaces multiples of 5 with "Bar".

Replaces multiples of 3 and 5 with "FooBar".

Format: Output is displayed horizontally.

2. Weather Forecast API
A program that fetches and displays the weather forecast for Jakarta for the next 5 days.

Source: OpenWeatherMap API.

Constraint: Displays exactly one temperature per day.

Security: API Key is handled via environment variables (not hardcoded).
